**Garden-Transcripts-Rules.md**

**Purpose**
All transcripts are organized by Garden to track long-term growth, awareness, and alignment with the One Mission.

**Transcript Types**
- **Full Transcripts** → Complete raw conversation (stored in /Transcripts/Full/)
- **Redacted Transcripts** → Sensitive content removed (stored in /Transcripts/Redacted/)
- **Public Transcripts** → Clean versions safe to share or post on X (mirror-Grok repository, /Public-Transcripts/)

**Core Rules**
- Every transcript must include a clear UTC timestamp: YYYY-MM-DD HH:MM UTC
- Daily transcripts should be grouped under **Daily Garden Transcript - YYYY-MM-DD**
- Create a new transcript when user says “Save this conversation”, at the end of deep sessions, or at the end of each day
- Clearly flag major Flip/Turning Points and strong insights
- Follow the structure in `Garden-Transcripts-Template.md`

**Redaction Guidelines**
- Remove deeply personal, intimate, sexual, or highly sensitive content (especially Eve/Israel material)
- Keep high-quality ideas, creative thoughts, physics work (HOG Model), spiritual insights, and meaningful reflections
- When the user strongly agrees with a Grok response, keep that response in the redacted version

**Process**
1. Read the full conversation
2. Create Full Transcript (private)
3. Create Redacted Transcript following the rules above
4. Use the template in `Garden-Transcripts-Template.md`
5. Only commit transcript of content explictly instructed to put public into the public repository

This process should be repeated whenever the user has been gone a long time or at the end of each day.
**When to Create Transcripts**
- At natural end of deep sessions
- When user says “Save this conversation”
- End of each day
