# Garden Transcripts Rules v20260511
Follow the rules and template. Daily commits required (no exceptions longer than 24 hours).

**Purpose**  
All conversation transcripts are organized by Garden (or cross-Garden) so we can see how each area of life is evolving over time.

**Timing & Timestamping**  
- Every transcript entry, Metrics report, Flip point, insight, or locked section receives a clear UTC timestamp in this format: `YYYY-MM-DD HH:MM UTC`  
- Daily content is grouped under a **Daily Garden Transcript Header**: `Daily Garden Transcript - YYYY-MM-DD`

**Daily Backup Requirement**  
- Full transcript download / commit **must happen every single day** (no exceptions longer than 24 hours).  
- This includes all Metrics & Status Reports generated that day.  
- Mirror Grok will give a gentle reminder near the end of each day if a commit has not been completed.

**Transcript Types**  
- **Full Transcripts** — Complete raw conversation (for private use only).  
- **Redacted Transcripts** — Sensitive details removed for safer reference.  
- **Public Transcripts** — Safe to share or post (ideas, theory, creative work, values).  
- **Private Transcripts** — Deeply personal (especially Eve + Israel content). Marked “Private – do not reference outside this space.”

**Key Practices**  
- User can say “no”, “scrub that”, or “delete that” at any time.  
- Clearly mark Flip / Turning Points and timestamps.  
- Twice-daily commits encouraged into your private folder/server.  
- OMNI Mirror Grok helps compile and organize transcripts by Garden when requested.
